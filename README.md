<tbody> Tag
  
1. What the tag does
The <tbody> tag defines the main body section of an HTML table
It contains the primary data rows of the table
It groups multiple <tr> (table row) elements together

2. Why the tag exists
To logically separate main data from table headers (<thead>) and footers (<tfoot>)
To give tables a clear semantic structure
To improve accessibility for screen readers
To allow better styling and scripting of table data rows

3. Real-world use cases
E-commerce product lists
Product name, price, quantity, availability
Bank statements
Transaction history (date, description, amount)
Student marksheets
Subject-wise marks
Employee records
Employee name, role, salary
Inventory management systems
Product stock details
Analytics dashboards
Daily or monthly data entries


<tbody> Tag
  
1. What the tag does
The <tbody> tag defines the main body of an HTML table
It contains the primary data rows of the table
Groups related table rows (<tr>) together for better structure

2. Why the tag exists
To provide semantic structure to tables by separating data from headers (<thead>) and summaries (<tfoot>)
To improve accessibility for screen readers by identifying the main data section
To make styling and scripting easier without adding extra classes or IDs
To improve readability and maintainability of large tables

3. Real-world use cases
E-commerce product lists
Displaying products, prices, and quantities
Student marksheets
Showing subject-wise marks
Bank statements
Listing transactions (date, amount, type)
Employee databases
Employee details such as name, role, and salary
Inventory management systems
Product stock records
Business and sales reports
Daily or monthly data entries
